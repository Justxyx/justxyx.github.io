## 外键

- 已创建的表，增加外键约束

```sql
ALTER TABLE camera ADD CONSTRAINT `fk_2` FOREIGN KEY (group_id) REFERENCES groups(group_id);

```


## 主键

- 对现有的列，更改其为主键

~~~sql
ALTER TABLE img_info ADD CONSTRAINT img_info_id PRIMARY KEY(img_info_id);
~~~
