# 字节流 OutputStream / InputStream

**万物皆可字节流** `OutputStream`

## 1. 文件输出流

```java
            File file = new File("temp1.log");
            FileOutputStream fos = new FileOutputStream(file);
            fos.write("你好啊".getBytes());
            fos.close();
```
