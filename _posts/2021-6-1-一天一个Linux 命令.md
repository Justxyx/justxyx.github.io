### zip 压接与解压

~~~java
// 将指定目录/tmp压缩成test.zip文件
zip -r test.zip tmb/

// 将压缩文件test.zip在指定目录/tmp下解压缩
unzip -o test.zip -d tmp/
~~~

### 删除文件夹
~~~java
//删除文件夹
rm -rf dirname
~~~

### 查看nvidia显卡显存情况
~~~java
// 查看显卡情况
nvidia-smi

// 杀掉显存占用程序
kill -9 pid

~~~

### 查看文件大小
可以使用ls -ll、ls-lh命令进行查看，当使用ls -ll，会显示成字节大小，而ls- lh会以KB、MB等为单位进行显示，这样比较直观一些。
